# Signal Playground Nano

A compact CR2032-powered RGB LED learning PCB made for Madeline.

This board demonstrates simple electronics concepts through a playful, hands-on layout:

- Common-anode RGB LED wiring
- Current-limiting resistors
- Momentary switch control
- Global brightness boost
- Coin-cell power
- Cute silkscreen art for a kid-friendly learning board

## Features

- CR2032 battery powered
- RGB LED with individual red, green, and blue buttons
- BRIGHT button that bypasses a series dimming resistor
- Compact rounded PCB
- Kid-friendly silkscreen labels and artwork
